# Simple Webpage Project

A clean, responsive static website built for GitHub Pages.

## Files

- `index.html` – main webpage
- `css/style.css` – styles for layout and design
- `js/script.js` – small interactive button script

## Usage

1. Open `index.html` in your browser to preview locally.
2. Create a GitHub repository and push these files.
3. Enable GitHub Pages from the repository settings to host the site.

## Notes

- The page is mobile-friendly and easy to customize.
- Replace the copy and colors to make it your own.
